# GV Timer - German Volume Training

A single-page web application for tracking German Volume Training (10 sets of 10 reps) workouts.

## Features

- **Set Counter**: Tracks the number of completed sets (0-10)
- **1-Minute Rest Timer**: Automatically starts a 60-second countdown timer after each completed set (sets 1-9)
- **Completion Message**: Displays a celebration message when all 10 sets are completed
- **Reset Button**: Resets the counter and timer to initial state at any time

## Usage

Simply open `index.html` in any modern web browser. No build process or dependencies required.

1. Click "Complete Set" after finishing each set
2. Rest for 1 minute while the timer counts down
3. Repeat until you complete all 10 sets
4. Use "Reset" to start a new exercise

## German Volume Training

German Volume Training (GVT) is a high-volume training method that typically involves performing 10 sets of 10 repetitions with the same weight, with 60 seconds of rest between sets.
